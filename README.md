# Parcel-React
零配置React开发模板，基于Parcel全新构建



> 使用说明
```bash
$ npm install -g parcel-bundler
创建一个 package.json
$ npm init -y
```

> 安装相关依赖
```bash
$ npm install
```

> 开发调试
```bash
$ npm run dev
```

> 正式构建
```bash
$ npm run build
```

>目录结构
```js
├── dist                构建结果
│   └── index.html
├── index.html          首页文件
├── node_modules
├── package.json
```
